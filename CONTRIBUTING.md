# Contributing

Thank you for taking a look at this project.
If you want to help, small improvements are welcome.

## Before you start
- For bugs / questions / feature requests: open a GitHub Issue.
- For security issues: please do not open a public issue. See SECURITY.md.

## Pull requests
- Please link an Issue.
- Keep changes small and focused.
- Explain what changed and why.
