# Code of Conduct

We want this project to be a welcoming place to collaborate.

## Expected behavior
- Be respectful and constructive.
- Assume good intent; discuss ideas, not people.

## Unacceptable behavior
- Harassment, discrimination, or personal attacks.
- Threats or doxxing.

## Scope
Applies to all project spaces (issues, pull requests, discussions, and related channels).

## Reporting
For security vulnerabilities, follow SECURITY.md.
