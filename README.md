# NoXmit AI

NoXmit AI is a non-profit research project focused on privacy-preserving, local-only AI inference that does not transmit user inputs, designed for Tor-compatible static web deployment.

The project aims to evaluate whether such designs can provide meaningful privacy in real-world use, while also assessing their practical advantages and limitations.

## Model
The initial model used in this project is Granite 4.0 350m in GGUF format.

It is included for local inference in the client-only prototype.
The model package also includes the upstream LICENSE, NOTICE, and MODEL.md files under models/granite-4.0-350m/.
